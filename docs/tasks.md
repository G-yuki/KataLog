## 設計メモ

### 地図共有機能
- 方式: publicToken（UUIDv4）を pairs コレクションに保存
- 公開ページ: `katalog-jp.web.app/share/{token}`
- Firestore rules: `/shares/{token}` に publicRead 許可
- Maps JS API embed で行った場所をピン表示
- コスト: $7/1,000 ページロード（Maps JavaScript API）
