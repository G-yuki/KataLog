# CLAUDE.md

## コミットメッセージ

`Co-Authored-By:` 行は不要。コミットメッセージに含めないこと。

## 修正指示を受けたときのルール

1. **即時記録**: 指示を受けたら実装より先に `docs/tasks.md` のタスク表に追記する。
   - 理由: コンテキスト圧縮でメモリが削除されると、指示や経緯が消える可能性があるため。
2. **方針共有**: コードの修正・実装に入る前に、修正方針（何をどう変えるか）をユーザーに提示して確認を得る。

## 危険なgit操作

### `git filter-repo` は絶対に使わない

`git filter-repo --path <dir> --invert-paths` は対象ディレクトリを git 履歴から完全削除するだけでなく、**ワーキングツリー（実ファイル）も削除する**。

- 2026-07-05 発生: `git filter-repo --path docs --invert-paths --force` を実行したところ、ワーキングツリーの `docs/` 全ファイルが削除された。
- 復旧に JSONL ツール履歴を手動で掘り起こす作業が必要になった。

**代替手段（docs を git 管理から外す場合）：**
1. `.gitignore` に `docs/` を追加（すでに設定済み）
2. git 履歴から過去コミットを消したい場合は `git rm -r --cached docs/` のみ実行する（ワーキングツリーは保持される）
